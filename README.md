# Predictive Urban Growth Modeling for Real Estate Investment

## 📌 Project Overview

This project uses Data Analytics and Machine Learning to identify real estate locations with high growth potential. By analyzing historical data such as property prices and demand trends, the model predicts future price growth and helps investors identify promising areas for investment.

The project demonstrates how data-driven insights can improve decision-making in the real estate sector.

---

## 🎯 Objective

To analyze real estate trends and predict future urban growth areas using historical price and demand data.

The system:

* Analyzes price growth patterns
* Identifies high-demand locations
* Predicts future property prices
* Visualizes trends using charts and heatmaps

---

## 🛠 Tools & Technologies Used

* Python
* Pandas (Data analysis)
* Matplotlib (Data visualization)
* Seaborn (Heatmap visualization)
* Scikit-learn (Machine Learning – Linear Regression)
* HTML & CSS (Frontend for visualization)

---

## 📊 Dataset Information

The dataset contains historical real estate data from 2018 to 2023.

Columns used:

* Location → Area name
* Year → Year of record
* Price → Average property price
* Demand → Buyer demand score

---

## ⚙ How the Model Works

### Step 1 – Data Collection

A sample dataset is used containing price and demand values for multiple locations.

### Step 2 – Data Analysis

Price trends are analyzed to identify growth patterns across different areas.

### Step 3 – Data Visualization

Charts are created for better understanding:

* Line Graph → Shows price trends over time
* Bar Chart → Shows total price growth by location
* Heatmap → Shows demand variation across years

### Step 4 – Machine Learning Prediction

Linear Regression algorithm is used to predict future property prices based on past trends.

The model predicts prices for the year 2025.
