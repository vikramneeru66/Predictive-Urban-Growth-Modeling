Predictive Urban Growth Modeling Dashboard

Analyzing and forecasting real estate trends to find high-growth investment opportunities.

Project Description

Welcome to our interactive data analytics dashboard. This project analyzes historical real estate data (2018-2023) across various urban locations, evaluating key metrics like pricing trends and buyer demand.

Using simple machine learning (Linear Regression), we have projected these historical trends into the future to identify which neighborhoods hold the highest growth potential for investors by the year 2025.
